# HolidayBot Telegram Wishlist & Birthday Manager

HolidayBot - Telegram-бот для хранения и управления списком подарков и отслеживания дней рождения друзей. Проект реализует логику регистрации пользователей, ведения вишлистов и взаимодействия через инлайн-кнопки Telegram.

## Основные возможности
* Регистрация и обновление профиля по Telegram-тегу  
* Указание и изменение даты рождения  
* Добавление подарков в формате «название ссылка»  
* Просмотр и удаление собственных подарков  
* Добавление и удаление друзей по Telegram-тегу  
* Просмотр вишлистов друзей  
* Автоматическая очистка устаревших сообщений и логирование всех действий  

## Архитектура проекта
Многослойная структура:
* Логика работы с Telegram API через long polling  
* Обработка сообщений и callback-запросов  
* Генерация текстового интерфейса и inline-кнопок  
* Слой работы с PostgreSQL через JDBC (пользователи, друзья, подарки)  
* Валидация данных и парсинг пользовательского ввода  
* Файловое логирование всех взаимодействий  

## Принцип работы
1. Пользователь запускает `/start` и выбирает раздел: профиль или друзья  
2. В профиле указывает дату рождения и добавляет подарки сообщениями формата  
3. Бот сохраняет данные в базе и отображает список подарков с кнопками для удаления  
4. В разделе друзей можно добавлять и удалять пользователей по тегу `@username`, а также просматривать их подарки  

## Стек технологий
**Язык:** Java  
**База данных:** PostgreSQL  
**Telegram API:** Pengrad Java Telegram Bot API  
**Технологии:** JDBC, Collections API, Regex, File I/O  

## Структура базы данных
* `users (telegram_tag, birth_date)`  
* `gifts (gift_id, telegram_tag, name, link)`  
* `friendships (telegram_tag_user, telegram_tag_friends)`  

## Запуск
1. Установить PostgreSQL и создать базу данных `holidayDB`  
2. Создать таблицы по структуре выше  
3. Ввести параметры подключения (логин, пароль, хост) в `Postgre`  
4. Добавить токен Telegram-бота в `HolidayBot.java`  
5. Собрать и запустить проект
